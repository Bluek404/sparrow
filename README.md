# sparrow

匿名版

## 运行时依赖

- libpq >= 9.4.0
- libssl
- libevent
- libunwind

## 运行

设置环境变量 `SP_URI` 为 PostgreSQL URI

例如`SP_URI="postgres://postgres:password@127.0.0.1:5433/sparrow"`

然后执行 `./sparrow init` 初始化数据库
